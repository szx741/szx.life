最大公约数

使用辗转相除法

定理：两个整数的最大公约数等于其中较小的那个数和两数相除余数的最大公约数。最大公约数（Greatest Common Divisor）缩写为GCD。

gcd(a,b) = gcd(b,a mod b)

const gcd = (a, b) => {
    if (b === 0) return a;
    return gcd(b, a % b);
};

另：最小公倍数 = 两数乘积 / 最大公约数乘积
